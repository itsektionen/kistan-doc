# Built-in addons

***Location:*** `builtin`

## Default

***Location:*** `builtin/default`


Currently does nothing, but should be in active
addons to allow future updates to have default
addons.

## Make Scripts Global Scope

***Location:*** `builtin/compatibility/MakeScriptsGlobalScope`

Makes script exist in the global scope, see
[Script Compatibility](./../technical/compatibility.html#global-scope-scripts)
